# RedisModel - a simple OHM (object hash mapper) for Redis

### Features
- Easy API
- Getters/Setters
- Model class and instance methods
- Defaults
- ID generation

### Install
`npm install rmodel`

### How to Run Tests
1. Install `coffeegulp` (`npm install -g coffeegulp`)
2. `coffeegulp test` or `coffeegulp test watch`
